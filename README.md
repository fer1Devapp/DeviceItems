# DeviceItems 📱

App Android que consume una API REST y muestra un catálogo 
de dispositivos móviles con precio, marca y tipo.

## Screenshots
![Catálogo de dispositivos](https://raw.githubusercontent.com/fer1Devapp/DeviceItems/main/deviceimg.png)

## Funcionalidades
- 📋 Listado de dispositivos consumiendo API REST
- 💰 Visualización de precio por dispositivo
- 🏷️ Tipo y marca de cada dispositivo
- 🔄 Carga de datos en tiempo real

## Tecnologías
- Kotlin
- Jetpack Compose
- Android Studio
- REST APIs

## Cómo correrlo
Clona el repositorio y ábrelo en Android Studio.
